# 💬 Real-Time Chat Application

## 🚀 Overview
This is a real-time chat application that enables users to send and receive messages instantly. Built using Node.js and Socket.IO, it ensures low-latency communication.

---

## 🔥 Features
- Real-time messaging using Socket.IO  
- JWT-based authentication  
- Secure user sessions  
- Redis caching for faster data access  
- Handles multiple concurrent users  

---

## 🛠 Tech Stack
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: PostgreSQL  
- Caching: Redis  
- Real-time: Socket.IO
